MarkLeft

How does it works?

MarkLeft is a simple markup language to write beautiful text using variables. On you left there is a text area that represents the template to render and on your right there is the result of the rendering process. There are some variables that are replaced automatically by the editor:

  • To place the current date in dd/mm/yyyy format use $now-date$
  • Using $now-datetime$ you can replace the current date time in dd/mm/yyyy hh:MM:ss
  • The location of the current window can be get using $location.href$ variable.
